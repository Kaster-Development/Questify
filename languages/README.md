# Questify - Translations

This directory contains all translation files for the Questify plugin.

## Available Languages

- **German (de_DE)** - Deutsch
- **English (en_US)** - English

## File Structure

### POT File (Template)
- `wp-faq-chat.pot` - Template file with all translatable strings

### PO Files (Translations)
- `de_DE.po` - German translation (source)
- `en_US.po` - English translation (source)

### MO Files (Compiled)
- `de_DE.mo` - German translation (compiled, loaded by WordPress)
- `en_US.mo` - English translation (compiled, loaded by WordPress)

**Note:** The .mo files are automatically generated by WordPress or can be created with tools like Poedit.

## Adding a New Translation

### With Poedit (recommended):

1. Open `wp-faq-chat.pot` with [Poedit](https://poedit.net/)
2. Select "Create New Translation"
3. Choose the target language
4. Translate all strings
5. Save as `{language_code}_{COUNTRY_CODE}.po` (e.g., `fr_FR.po` for French)
6. Poedit automatically creates the .mo file

### Manually:

1. Copy `wp-faq-chat.pot` and rename it to `{language_code}_{COUNTRY_CODE}.po`
2. Update the header with language information
3. Translate all `msgstr ""` entries
4. Compile to .mo with `msgfmt`:
   ```bash
   msgfmt -o {language_code}_{COUNTRY_CODE}.mo {language_code}_{COUNTRY_CODE}.po
   ```

## Updating the POT File

When new translatable strings have been added:

### With WP-CLI:
```bash
wp i18n make-pot /path/to/wp-faq-chat /path/to/wp-faq-chat/languages/wp-faq-chat.pot
```

### With Poedit:
1. Open an existing .po file
2. Select "Catalog" → "Update from sources"

### Manually:
Add new entries to the .pot file in this format:
```
#: file.php:123
msgid "Original text"
msgstr ""
```
